This IT talk is abnout the overview of ICT jobs.

In the IT realm, the Android Developer focuses on feature implementation, code enhancement, and bug resolution . Junior UI/UX Designers contribute to research, wireframing, and prototyping using Figma. QA Testing Analysts prioritize documentation, testing, and bug reporting. Q:Why do we use a cooler in our design for the dispenser?
A:We used the cooler to make sure the wet food wont go bad.

Q:If you are far away from the dispenser,how do you control it using Wi-fi?
A:The user will make sure the dispenser is connected to the Wi-fi where it can access the internet. The user's device must be accessed to the internet to so the input given through the application will go through our server and the dispenser will receive it.

Q:What will be the biggest hurdle in your journey?
A:The biggest hurdle for us would be creating and maintaining the server especially if we want to go global.

We should find a way on how to create a cost-effective server if we decided to go globally,the performance will only drop slightly due to less complicated input and output transmitted between two devices. We should also make sure the cooler will be effective in maing sure the wet food wont go bad and also upgrade the performance of the dispenser.
